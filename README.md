# C# Windows Forms - BMI 計算機

## 📝 專案簡介
本專案為一個使用 C# Windows Forms 開發的基礎 BMI (Body Mass Index) 計算機。
[cite_start]使用者輸入身高與體重後，程式會自動計算出 BMI 值，並根據衛福部標準判斷所屬的體位區間（如：體重過輕、健康體位、過重、肥胖等）[cite: 467]。
為了提升使用者體驗與介面美觀，本專案額外實作了「輸入防錯機制」以及「依據體位動態變更結果文字背景顏色」的功能，並新增了清除按鈕方便連續計算。

## ✨ 核心功能
* [cite_start]**BMI 數值計算**：精準計算至小數點後兩位 [cite: 476, 480]。
* [cite_start]**防呆與錯誤處理**：驗證使用者輸入是否為空值或非數字，並阻擋小於等於 0 的無效身高體重輸入 [cite: 485, 497][cite_start]，透過 `MessageBox` 彈出警告提示 [cite: 508]。
* [cite_start]**動態體位標示**：透過陣列管理體位結果與對應顏色 [cite: 535, 536]，計算後會動態改變 Label 的背景顏色以警示使用者。
* **介面優化（加分項目）**：調整了字體對比度，並新增「清除」按鈕，一鍵重置所有輸入框與結果。

## 🚀 執行說明
1. [cite_start]確認電腦已安裝 **Visual Studio 2022** 與 **.NET Framework 4.7.2**（或以上版本）[cite: 47]。
2. 複製此儲存庫或解壓縮專案資料夾。
3. [cite_start]點擊 `BMI計算機.sln` 方案檔開啟專案 [cite: 43]。
4. [cite_start]在 Visual Studio 上方工具列點擊「開始」或按下 `F5` 鍵編譯並執行程式 [cite: 62]。
5. 於畫面中輸入您的「身高 (cm)」與「體重 (kg)」，點擊「計算」即可查看結果。
